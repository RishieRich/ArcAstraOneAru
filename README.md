# ArcAstraOneAru

ArcAstraOneAru is the project repository for the Astra launch work.

## Getting started

Clone the repository and switch to the main branch:

```bash
git clone https://github.com/RishieRich/ArcAstraOneAru.git
cd ArcAstraOneAru
git checkout main
```

Project setup and usage details will be added as the application is developed.
